# Geometric Area Calculator (C++)

A C++ mini project for my learning process. 
This project is a simple console application that calculates the area of various geometric shapes (Square, Rectangle, Triangle, Circle) based on user input.

## Features
* **Function Overloading:** Demonstrates the use of multiple functions with the same name but different parameters.
* **Error Handling:** Validates user input and provides warnings for invalid options.
* **Mathematical Operations:** Uses the `cmath` library for power and constant calculations.

## How to Run
1. Ensure you have a C++ compiler installed (GCC, Clang, or MSVC).
2. Compile the `main.cpp` file:
   ```bash
   g++ main.cpp -o area_calculator
